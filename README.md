# 🎬 Streaming Frontend

Client application for the Streaming Web App.
Built with React / Next.js + TailwindCSS.

## 🛠 Tech Stack

- React / Next.js
- Tailwind CSS
- Axios
- JWT Auth
- Docker

## ⚙️ Environment Variables

NEXT_PUBLIC_API_URL=

## 🚀 Run Locally

npm install
npm run dev
